Kryven

The Kryven project is an innovative crypto web application dedicated to sharing trading signals and automatically executing orders. The platform allows users to access reliable trading signals generated by experienced traders or analytical systems, helping them make better decisions in the crypto markets. Through integration with the KuCoin API, the application enables users to trade directly on their own accounts in a secure and transparent way, without intermediaries. Users retain full control of their funds while benefiting from fast and optimized execution. Designed to be intuitive and high-performing, the platform emphasizes security, risk management, and performance tracking. This project demonstrates my ability to design complex web solutions in the crypto space, combining automation, reliability, and a modern user experience.

Stack used for this project:

React.JS
Express.JS
Mongo db

Kaypic clone

The Kaypic project is a collaborative web application designed to help teams work together efficiently across recurring seasons or cycles. The platform enables users to create and manage teams, organize seasonal projects, and collaborate in real time through an integrated chat system powered by SignalR. This real-time communication layer ensures instant message delivery, live updates, and a smooth collaborative experience without page reloads. The application focuses on scalability, responsiveness, and a clear team-based structure, allowing users to stay synchronized throughout each season. With an intuitive interface and robust real-time features, this project demonstrates my ability to build collaborative web solutions, implement real-time technologies, and design systems that support teamwork, communication, and long-term project organization.

Stack used for this project:

Blazor
Asp .Net core 9
Microsoft sql server

"Hotel res"

Hotel Res is a full-stack Android application designed to streamline hotel booking and reservation management. The mobile app, built in Kotlin, provides users with an intuitive interface to browse available rooms,create bookings, and manage their reservations seamlessly. The application communicates with a robust backend API developed using FastAPI and SQLAlchemy, ensuring high performance, scalability, and clean data handling.

The backend implements secure authentication using access and refresh tokens, along with role-based access control to manage different user permissions. Core features include room management, booking creation, reservation tracking, and administrative tools for managing hotel data. This project highlights my ability to develop end-to-end mobile solutions, integrate Android applications with modern RESTful APIs, and design secure, well-structured backend systems for real-world use cases.

Stack used for this project:

Kotlin (Android)
fastapi (Backend)
SQLalchemy
Docker & Docker compose
